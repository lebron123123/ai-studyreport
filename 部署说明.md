## 二、创建 D1 数据库（免费，5分钟）

1. Cloudflare 后台左侧 → **存储和数据库** → **D1 SQL 数据库** → **创建数据库**，名字随意（如 `fs-db`）
2. 创建后进入该数据库 → **控制台（Console）**标签 → 把 `schema.sql` 文件里的全部内容粘贴进去 → 执行。看到成功提示即表——`users` 和 `projects` ——建好了
3. 回到你的 **Pages 项目 → 设置 → 绑定** → 添加绑定 → 类型选 **D1 数据库** → **变量名称填 `DB`**（一字不差）→ 选择刚建的数据库 → 保存

## 三、环境变量（新增一个）

设置 → 环境变量，新增：

| 变量名 | 值 | 说明 |
|---|---|---|
| `SESSION_SECRET` | 一串随机长字符（30位以上，比如键盘乱敲混合大小写数字） | 登录令牌的签名密钥，设完不要改，改了所有人都要重新登录 |

已有的继续沿用，但含义变了：
- `ACCESS_CODE`：现在是**注册邀请码**——只有拿到它的人才能注册账号。想换随时改。
- `DEEPSEEK_API_KEY`、`USAGE_KV`：不变。限额现在是**按用户**每天120次（改 generate.js 里的 `USER_DAILY_LIMIT`）。

## 四、重新部署并验证

上传全部文件 → 等部署完成 → 打开网站应看到**登录/注册**弹窗：
1. 切到"注册"，填用户名、密码、邀请码（`ACCESS_CODE` 的值）→ 注册即进入
2. 建一个项目、填点信息 → 顶部用户栏出现"已保存到云端"
3. 换台电脑（或无痕窗口）登录同一账号 → 点"我的项目" → 能看到并打开刚才的项目

## 五、本版产品能力

- **注册/登录**：密码经加盐PBKDF2哈希存储（数据库里看不到明文），登录态30天，令牌HMAC签名防伪造
- **云端项目库**：每人只能看到自己的项目；编辑自动云端保存（1.2秒防抖）；顶部用户栏可打开"我的项目"列表（打开/删除）、新建项目、退出
- **按用户限额**：AI生成从"全站共享200次/天"改为"每用户120次/天"，谁用超了只限谁
- **本地草稿保留**：浏览器本地存档依然工作，作为断网时的兜底
